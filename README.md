# portfolio-dashboard

社内向けシステム鳥瞰図・ダッシュボードプラットフォームのリポジトリです。

## 目的

**本リポジトリは個人の学習目的で作成したものです。**

ITポートフォリオの可視化や、システム構成・プロジェクトの一覧表示など、社内で利用するダッシュボードの試作・練習を行っています。
